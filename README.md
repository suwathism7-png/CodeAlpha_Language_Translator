# AI Language Translation Tool

A modern, responsive web application built with Python, Flask, and the `deep-translator` library that allows users to translate text across multiple languages smoothly. This project was developed as part of the CodeAlpha Software Engineering Internship.

## 🚀 Features
* **Language Auto-Detection:** Automatically identifies the source language of the entered text.
* **Multi-Language Support:** Translates text into Spanish, French, German, Hindi, Arabic, Japanese, and Chinese.
* **Responsive UI:** Built with a clean, dual-panel grid using Tailwind CSS.
* **One-Click Copy:** Includes a native "Copy to Clipboard" utility feature for quick extraction of translated text.
